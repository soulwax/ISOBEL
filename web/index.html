ISOBEL: Self-Hosted Discord Music Bot

ISOBEL is an open-source Discord music bot built for reliable playback, clean audio quality, and practical server-level controls. It includes caching, queue tools, and a web dashboard for configuration.

Add to Discord | View on GitHub | Songbird API

Core Features

  • High-quality audio streaming with smart queue management
  • Per-server configuration through a Discord-authenticated dashboard
  • Support for direct MP3 playback, YouTube URLs, and livestream audio
  • Open-source, self-hosted architecture for full ownership and control

Quick Setup

Clone the repository, configure environment variables, install dependencies, and start the bot and web interface to manage settings.
